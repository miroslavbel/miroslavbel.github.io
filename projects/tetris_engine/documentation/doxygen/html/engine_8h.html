◆ GetNextTetrominoFunction

typedef void GetNextTetrominoFunction(NextTetromino *nextTetromino)

Генерирует следующее тетрамино.

NextTetromino::size может принимать значения [2 .. tetrominoMaxSize]. А тетрамино в NextTetromino::pixels должно быть смещено к началу координат (координаты 0:0).

Функция обязана обработать каждый пиксел в NextTetromino::pixels даже если у сгенерированного тетрамино NextTetromino::size меньше, чем tetrominoMaxSize.

Parameters
[out]nextTetrominoструктура, в которую функция должна записать cгенерированное тетрамино
Examples
\examples\gdi_example.c.

◆ TetrominoPixel

typedef uint8_t TetrominoPixel

Пиксел тетрамино.

Значение 0 означает, что пиксел свободен. Любое другое значение означает, что пиксел занят. Разные ненулевые значения можно использовать, например, для кодирования цветов тетрамино.

Игра создана, но ещё не было ни одного тика.

Инициализирует игру.

В Game::status записывает initGameStatus.

Warning
Функция аллоцирует массивы для следующего и активного тетрамино, но не инициализирует их. Для полной инициализации требуется вызвать функцию startGame.
Parameters
[in]widthширина игрового стакана. Должна быть больше или равна tetrominoMaxSize
[in]heightвысота игрового стакана. Должна быть больше или равна tetrominoMaxSize
[in]maxScoreмаксимально возможное количество очков
[in]getNextTetrominoFunctionфункция, генерирующая следующее тетрамино
[in]getScoreAddendFunctionфункция, вычисляющая количество очков, которое игрок заработал за заполнение строк
Returns
  • 1) NULL в случае ошибки (нехватка памяти);
  • 2) указатель на структуру.
