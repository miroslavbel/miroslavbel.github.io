◆ gameField

TetrominoPixelArray const gameField

Одномерный массив пикселов тетрамино игрового стакана.

Note
Длина одномерного массива равна width * height. Для доступа как к двухмерному массиву используйте макрос flatArrayAs2D. Ширина и высота двухмерного массива равна width и height соответственно.
